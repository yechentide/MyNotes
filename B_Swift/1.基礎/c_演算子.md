# 演算子



### 演算子

* 算術演算子：　`+` `-` `*` `/` `%`

* 論理演算子：　`&&` `||` `!`

* 比較演算子：　`>` `<` `>=` `<=` `==` `!=`

* 代入演算子：　`=`

* 複合代入演算子：　`+=` `-=` `*=` `/=` `%=` `&&=` `||=`

  Swiftでは`++`(インクリメント)と`--`(デクリメント)がない

* 三項演算子：　` ? : `

  ```swift
  条件式 ? trueの場合の式 : falseの場合の式
  ```

* 範囲演算子：　`..<` `...`

  ```swift
  1..<3		// 1 <= x < 3  となる範囲
  1...3		// 1 <= x <= 3 となる範囲
  ```

* ビット演算子：

  2進数は`0b`, 16進数は`0x`から始まる

  1. bitシフト：　`<<` `>>`
  2. bit積、bit和、排他的bit和、bit否定：　`&` `|` `^` `~`



### 演算子前後の空白の統一

`a+b` と `a + b`は大丈夫だが、`a+ b`や`a +b`はエラーになる
