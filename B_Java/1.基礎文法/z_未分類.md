# 未分類



## 標準クラスのメソッド



### Math

* `Math.sqrt(x)`：xの平方根
* `Math.abs(x)`：xの絶対値
* `Math.pow(a,b)`：aのb乗
* `Math.random()`：0.0 ~ 1.0の間の乱数



### 文字列

* `文字列.length()`
* `文字列.charAt(index)`
* `文字列.equals(str)`
* `文字列.isBlank()`
* `文字列.isEmpty()`
* `文字列.trim()`
* `文字列.repeat(n)`
* `文字列.replace(str1, str2)`
* `文字列.substring(startIndex, endIndex)`
