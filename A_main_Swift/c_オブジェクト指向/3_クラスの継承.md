# クラスの継承





## クラスの拡張



### クラスの継承

```swift
class クラス名: スーパークラス {
    // ...
}
```

* スーパークラスのメソッドの実行

  ```swift
  super.メソッド名()
  ```

* スーパークラスのメソッドの上書き

  ```swift
  override func メソッド名() -> () {}
  ```



### final

* クラス名の前に`final`をつけると、継承禁止
* メソッドの前に`final`をつけると、継承禁止
