● シェルと日常作業のコマンド
	・シェルの環境設定
		echo で文字列表示
		echo $HOME
		
		Linuxのコマンド = 組み込みコマンド + 外部コマンド
		bashの主な組み込みコマンド：
			alias		// エイリアスを作成したり、表示したりする
			echo		// 引数の内容を表示する
			exit		// シェルを終了する
			export		// 環境変数を設定する
			history		// コマンド履歴を表示する
			pwd			// カレントディレクトリのパスを表示する
			unalias		// エイリアスを削除する
		
		「type コマンド」　で、指定コマンドの保存場所がわかる
		
		変数にはシェル変数と環境変数の二つがある
		シェル変数の定義：	「変数名=値」
		「export 変数名」　で、シェル変数を環境変数にする
	
	・エイリアス機能・・・コマンドに別名をつける
		デフォルトのエリアンス：		「ll」=「ls -alF」	// ファイルの一覧表示
		「alias」で定義済みのものを表示
		「alias 別名='コマンド'」でエイリアスを設定

● テキスト処理コマンド
	・リダイレクト		>	>>		2>		2>>
	・内容の表示
		先頭だけ：		head [-行数] [ファイル名]
		末尾だけ：		tail [-行数] [ファイル名]
		行数を数える：		wc [オプション] [ファイル名]	// 行数、単語数、バイト数を表示
							-l		行数だけを表示
	・文字列検索：
		grep [オプション] 文字列パターン [対象ファイル]
			-n		行番号も表示
			-i		大文字小文字を区別せずに検索
			-c		マッチした行の行数だけを表示
			-v		マッチしなかった行だけを表示
	・正規表現で検索
		単語指定ではなく、いろんな条件で検索するときに役に立つ
		正規表現：
			.		// 任意の1文字にマッチする
			*		// 直前の文字の0回以上の繰り返しにマッチする
			^		// 行頭にマッチする
			$		// 行末にマッチする
			(|)		// | 前後いずれかの文字列にマッチする
		表記例：
			1.3			// 123, 1a3 など
			.*			// 0文字以上の文字列
			nba*		// nba, nbaa, nbaaa など
			^nba		// 行頭にある nba にマッチする
			^#			// 行頭が # で始まる行にマッチする
			^$			// 空行にマッチする
			(SR|AQ)		// SR AQ のいずれかにマッチする。"(SR|AQ)" の形にすべき
	
	・ファイルを検索する
		locate ファイル名
		find 検索するディレクトリ -name "ファイル名"
		find 検索するディレクトリ -atime +30		// 最終アクセス日が３０日以前の
			-atime		// 最終アクセス日で検索
			-amin		// 最終アクセスを分単位で指定して検索
			-mtime		// 最終更新日で検索
			-type f		// ファイルだけ
			-type d		// ディレクトリだけ
			-perm		// アクセス権で検索
	
● ネットワークコマンド
	・URLでダウンロード
		wget URL
		wget -O 保存したい別名 URL			// -O の O は大文字
			-q		// エラー以外もメッセージを非表示

● システム管理コマンド
	・プロセス一覧
		ps [オプション]
			axu		// 全体のプロセスを表示
	・プロセスやシステムの状態を監視する
		top
	・プロセスを終了させる
		kill [-シグナル] PID
				-KILL		// 強制終了
	・OSを確認する
		uname -a		// システム情報を表示する
	・ハードウェア構成を確認する
		CPU情報：
			cat /proc/cpuinfo
