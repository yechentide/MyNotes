# 基本型

## リテラル

### 論理リテラル

`true` `false`

### 数値リテラル

`123` `1.2` など

### 文字リテラル

`'a'` `'b'` など`'`で囲むもの

### 文字列リテラル

`"abc"` など`"`で囲むもの

## エスケープシーケンス

### よく使われるもの

| エスケープシーケンス |        効果        |
| :------------------: | :----------------: |
|         `\n`         |    ラインフード    |
|         `\r`         | キャリッジリターン |
|         `\"`         |   ダブルクオート   |
|         `\'`         |  シングルクオート  |
|         `\\`         |  バックスラッシュ  |
|         `\0`         |      null文字      |
|         `\t`         |        タブ        |
