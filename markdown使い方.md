# 段落と改行


## 段落
第1段落：段落を変えるには空行を一行以上いれればオッケー！

第2段落：ねっ？変わったでしょう。



第3段落：これだけ開けても大丈夫


## 改行
改行をするには最後の行に空白を2個以上入れる(スペースを最後に2回⇒)   
ほらっ変わった。
空白を入れないとエディタでは改行されていても改行されないので要注意。



# 強調表現
`*`1個で*斜体で強調*  
`**`2個で**太字で強調**  
`***`3個で***斜体&太字で強調***

フォントにMeiryoとか使ってると、斜体で表示されないので注意。  
まあ斜体はあんま使いどころないので太字で強調だけ覚えておきましょう。
